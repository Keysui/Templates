This is a **Next.js 16** website project featuring **three professional website templates**:
1. **Professional Template** - Legal firm website (Vanguard Legal Group)
2. **Aesthetic Template** - Wellness & MedSpa website (Luna Wellness)

## ğŸ“ Customization Guide

### Changing Colors:
Edit the color constants in each template file:
- **Professional:** `EM7`, `EM7A`, `EM7B` constants
- **Aesthetic:** Amber/gold color values
- **Modernist:** Cyan/slate color values

### Changing Content:
- Text content is directly in the template files
- Replace placeholder text with your actual content
- Update contact information, addresses, phone numbers

### Adding New Sections:
- Each template uses a tab/section system
- Add new sections by extending the tab array and adding corresponding JSX
